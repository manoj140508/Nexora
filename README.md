## Nexora

## The All-in-One Student Ecosystem

Nexora is a student productivity platform that combines academics, productivity, project management, career development, and AI-powered tools into a single ecosystem.

### Current Features

#### Authentication
- User Signup
- User Login
- Secure Dashboard Access
- Logout System

#### Dashboard
- Personalized Welcome Screen
- Academic Overview
- Quick Navigation

#### Academic Hub
- Subjects Manager
- Assignment Manager
- Attendance Tracker
- Notes Manager

### Technologies Used
- HTML
- CSS
- JavaScript
- Local Storage

### Upcoming Features
- GPA Calculator
- Study Planner
- Flashcards
- PYQ Manager
- Academic Analytics
- Nexora AI
- Career Hub Enhancements

### Vision
To create a single platform where students can manage academics, productivity, projects, career preparation, and collaboration.
